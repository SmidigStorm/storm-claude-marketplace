# pm-architecture

Manage tech stack and architecture decisions (ADRs).

## Commands

### `/architecture`

Interactive menu to:

- **Update tech stack** - Document technologies used
- **New decision** - Create an Architecture Decision Record
- **Review decisions** - List and manage existing ADRs

## Skills

### `adr`

Knowledge about Architecture Decision Records - when to create them, format, status lifecycle, and writing tips.

## Output

- Tech stack: `docs/strategy/tech-stack.md`
- ADRs: `docs/strategy/architecture/NNN-title.md`

## Installation

```
/plugin install pm-architecture@storm-claude-marketplace
```
