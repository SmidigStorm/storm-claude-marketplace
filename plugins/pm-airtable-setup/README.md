# pm-airtable-setup

Set up Airtable tables for PM plugins.

## Commands

### `/airtable-setup`

Creates the required tables in an Airtable base for:
- `pm-domain-knowledge` (Domain, Subdomain, Entity, Attribute, Process, Glossary)
- `pm-requirements` (Requirement, Rule, Example, OpenQuestion)

## Prerequisites

1. Airtable MCP server configured
2. Empty Airtable base ready
3. Token with write permissions

## Installation

```
/plugin install pm-airtable-setup@storm-claude-marketplace
```
