# pm-prioritization

Manage product backlog with MoSCoW prioritization.

## Commands

### `/backlog`

Interactive menu to:

- **Add item** - Add a new feature or fix
- **Prioritize** - Assign MoSCoW priorities (Must/Should/Could/Won't)
- **View** - See backlog grouped by priority

## Output

- Backlog: `docs/strategy/backlog.md`

## MoSCoW

- **Must** - Critical for next release
- **Should** - Important but not critical
- **Could** - Nice to have
- **Won't** - Not this time

## Installation

```
/plugin install pm-prioritization@storm-claude-marketplace
```
